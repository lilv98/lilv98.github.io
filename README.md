# lilv98.github.io

Personal academic homepage of Zhenwei (Joseph) Tang.

Static HTML site (no build step), served directly by GitHub Pages.

- `index.html` — home: bio, news, education, experience, service
- `publication.html` — selected publications
- `style.css` — styling
- `img/` — headshot and assets

Template adapted from the classic Jon Barron / Stephen Bach academic homepage layout.

## Local preview

```
python3 -m http.server 4321
```

Then open http://localhost:4321.
